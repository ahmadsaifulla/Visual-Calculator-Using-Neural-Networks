# Visual Calculator Using Neural Networks

## Overview

The application takes input from user in the form of drawing canvas, where the user draws 2 digits and one operator. The data is then send to the backend as 28x28 images and 2 models (one for operator and one digit) classifies the input accordingly and computes the expression and returns result back to the UI.


## Technology Stack

HTML, CSS for Web UI
Flask for python backend
CNN for classification of images.

## How to run

- Download and extract the given .zip file
- Create a virual enviroment and activate it.
- Run `pip install -r requirements.txt`
- Then finally run `flask run` to serve the web page.
